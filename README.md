# 🚗 CarBook - Araç Kiralama Sistemi

- Modern ve kullanıcı dostu bir araç kiralama platformu.
- Bu proje eğitim amacıyla geliştirilmiş olup ASP.NET Core MVC teknolojileri kullanılarak hazırlanmıştır.


# 🚀 Proje Hakkında

- CarBook, kullanıcıların araçları inceleyip rezervasyon oluşturabildiği, admin paneli üzerinden rezervasyonların yönetilebildiği kapsamlı bir araç kiralama sistemidir.

- Projede N Katmanlı Mimari (N-Tier Architecture) yapısı kullanılmıştır. Katmanlar arasında veri transferi için DTO yapıları tercih edilmiş, nesne dönüşümleri için AutoMapper kullanılmıştır.


# 🛠️ Kullanılan Teknolojiler
- ASP.NET Core MVC
- Entity Framework Core
- MSSQL
- N Katmanlı Mimari
- DTO Yapısı
- AutoMapper
- LINQ
- Bootstrap
- HTML / CSS / JavaScript


# ✨ Özellikler

# 👤 Kullanıcı Tarafı
- Araç listeleme
- Araç detay sayfası
- Rezervasyon oluşturma
- Rezervasyon onay sistemi
- İndirim kodu sistemi
- Dinamik veri listeleme
- Özel 404 hata sayfası


# 🔐 Rezervasyon Sistemi
- Kullanıcı rezervasyon oluşturduğunda doğrulama/onay kodu gönderilir
- Admin rezervasyonu onayladığında kullanıcıya:
- Rezervasyon onay mesajı
- İndirim kodu
gönderilir
- Rezervasyon onaylandıktan sonra araç otomatik olarak kilitlenir
- Aynı araç için başka kullanıcı rezervasyon oluşturamaz


# ⚙️ Admin Paneli
- Tüm CRUD işlemleri
- Araç yönetimi
- Rezervasyon yönetimi
- Veri listeleme ve düzenleme işlemleri

  
# 🏗️ Mimari Yapı
Projede aşağıdaki katmanlı yapı kullanılmıştır:

- Presentation Layer
- Business Layer
- Data Access Layer
- Entity Layer
- DTO Layer

# Bu yapı sayesinde:

- Kod okunabilirliği artırıldı
- Sürdürülebilir bir mimari oluşturuldu
- Katmanlar arası bağımlılık azaltıldı


# 📌 Teknik Detaylar

- Veriler DTO üzerinden taşınmıştır
- AutoMapper ile entity ve DTO dönüşümleri yapılmıştır
- Asenkron programlama yapısı kullanılmıştır
- CRUD işlemleri tamamen aktif şekilde çalışmaktadır
- Rezervasyon çakışmalarını önleyen kontrol mekanizması geliştirilmiştir
